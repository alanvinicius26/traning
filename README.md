# Treino Vivo

App web responsivo para montar e acompanhar treinos de academia em portugues do Brasil.

## O que esta incluido

- Rotinas semanais de 3 ou 5 dias por objetivo diario.
- Perfis de nivel iniciante, intermediario e avancado.
- Biblioteca essencial com 36 exercicios de academia completa.
- Animacoes ilustradas em SVG/CSS para cada padrao de movimento.
- Registro local de series, repeticoes, carga, observacoes e treinos concluidos.
- Historico salvo no navegador com `localStorage`.

## Como rodar

Instale as dependencias e inicie o servidor local:

```bash
npm install
npm run dev
```

Depois abra o endereco mostrado pelo Vite no navegador.

## Build

```bash
npm run build
```

Esta primeira versao nao usa login, backend ou sincronizacao em nuvem.
